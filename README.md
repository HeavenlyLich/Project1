# 🎮 Гра "Хрестики-Нулики" на Python (Tkinter)

Цей проект є навчальним застосунком, розробленим у межах курсу з програмування. Програма демонструє використання бібліотеки **Tkinter** для створення GUI, а також впровадження **Unit-тестування** (unittest, pytest).

## ✨ Особливості (Features)
* 🧩 **Класичний ігровий процес**: Гра 3x3 для двох гравців.
* 🛠 **Виправлена логіка**: Усунено баги з перевіркою діагоналей та перезаписом клітинок.
* 📢 **Індикація ходу**: Динамічне відображення поточного гравця (X або O).
* 🧪 **Тестове покриття**: Проект містить модульні тести для перевірки ігрової логіки.
* 📱 **Адаптивний інтерфейс**: Вікно програми автоматично підлаштовується під контент.

## 🚀 Як запустити
1. Переконайтеся, що у вас встановлено **Python 3.10+**.
2. Склонуйте репозиторій:
```bash
git clone https://github.com/HeavenlyLich/Project1.git
```
3. Перейдіть у папку проєкту:
```bash
cd Project1
```

4. Запустіть гру:
```bash
python main.py
```

## 🧪 Тестування
Для запуску тестів використовуйте pytest:
```bash 
pytest -v
```
